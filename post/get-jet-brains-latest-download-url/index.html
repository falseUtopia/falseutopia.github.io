2021-06-14

获取 Jet brains 系列最新下载链接

说明： 因为jetbrains官网有时慢的离谱，所以才有下面的分析

分析官网

官方下载页面地址：
https://www.jetbrains.com/zh-cn/idea/download/download-thanks.html?platform=windows

得到地址

获得一份下载详情的地址： https://data.services.jetbrains.com/products/releases?code=IIU%2CIIC&latest=true&type=release&build=&_=1623607052641

  • 参数分析
    • code ： 产品编码； 支持选项 IIU，IIC， 详情见下方分析
    • latest： 标识最新版本； 支持选项 true/false， 如果不带，返回历史所有版本
    • type ： 发布版本 ； 支持选项 release，且默认值是 release
    • build ： 暂时未知；
  • 返回结果分析
    • IIC：产品编码
      • date： 更新时间
      • type： 发布版
      • downloads： 下载地址
        • linux： 平台
          • link： 各个平台的下载地址

分析产品编码

产品编码分析：在源码中 全局搜索 关键字 IIU

快捷键 CtrlShiftF

最终下载地址获取

# https://data.services.jetbrains.com/products/releases?code={code}&latest=true&_=1623607052641

# code替换为对应的产品编码，如 IIU
curl -s https://data.services.jetbrains.com/products/releases?code=IIU&latest=true&_=1623607052641

# 配合jq解析json， 这是jq的安装方式 > https://stedolan.github.io/jq/download/

# linux
curl -s https://data.services.jetbrains.com/products/releases?code=IIU&latest=true&_=1623607052641 | jq '.IIU[0].downloads.linux.link'

# windows
curl -s https://data.services.jetbrains.com/products/releases?code=IIU&latest=true&_=1623607052641 | jq '.IIU[0].downloads.windows.link'
